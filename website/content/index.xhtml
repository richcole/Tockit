Welcome to Tockit

Welcome to the Tockit site. This site describes Tockit and its subprojects. Please choose one of the sections to find the descriptions for the different aspects of Tockit:

  • Tockit -- the generic framework
  • ToscanaJ -- a Java port of the classic FCA tool
  • Docco -- a personal document retrieval system
  • Score -- a web-based document retrieval system
  • CGXML -- mapping conceptual graphs into XML
