Docco Overview

Docco is a tool that should help you finding documents in your document collection. It first scans all documents for the appearance of words (called "indexing") and then lets you search for those keywords and some other information like the author of certain documents (called "querying"). After a query the results are displayed first in a diagram showing you all possible combinations. Once you select a certain node in this diagram, you will get a tree structure similar to the one used in standard tools like file dialogs or the Windows file explorer. From there you can open the documents in their default application by double-clicking or get more information in the summary pane at the bottom.

The online help is structured in the following sections:

Quick start
Gives you online what you really need to know.
Indexing
Tells you more about how indexes are created and how to manage them.
Query syntax
Gives you more detail on what you can ask the system.
View options
Explains the options in the view menu.
