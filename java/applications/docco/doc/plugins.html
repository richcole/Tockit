Plugins

Docco's functionality can be extended by plugins. Plugins are little program components which are installed into the "plugin" directory in the Docco installation. To install a plugin you have to extract it and copy the extracted folder into this directory. After restarting Docco you should be able to use the new functionality.

Docco plugins can be downloaded from the Tockit project's download pages, where you find the latest Docco version, too. At the time of writing the following plugins exist:

Indexing tools

pdfbox-plugin
allows you to index PDF documents
poi-plugin
allows indexing MS Word and MS Excel documents
multivalent-plugin
allows indexing PDF, DVI, UNIX man-pages and other formats

Graphic export

batik-plugin
allows exporting into SVG format
freehep-plugin
allows export into a number of graphic formats: PDF, (E)PS, EMF/WMF and others
